    white-space

    how to deal with empty space

    
                  white-space: normal;   // no br, no multi space, only wrap
                  white-space: nowrap;   // no br, no multi space, NO wrap
                  white-space: pre;      // yes br, yes multi space, but NO wrap
                  white-space: pre-line; // yes br, yes wrap, but NO multi space
                  white-space: pre-wrap; // yes br, yes wrap, yes space

    ime-mode

    to control input language. *IME = Input Method Editor
    But avoid using it, as it may be removed from web standards.
